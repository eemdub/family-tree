# Family tree

Project with Go and Neo4j.

## Keywords

[Go, Neo4j, Clean Architecture]

## Requirements

- Docker
- Docker-compose
- Go >= 1.11
- Neo4j

## Installation

`git clone https://github.com/larien/family-tree.git`

## Run application

### Option 1: docker-compose [unstable]

```bash
make compose
```

### Option 2: docker

```bash
make docker
```

## Running tests

```bash
cd backend/tests/
go test
```
